====== Тег {{i18n}} ======

===== Описание =====
переводит текст при помощи функции lmb_i18n и выводит его в браузер.




===== Синтаксис =====

<code xml>
{{i18n text="some text for translate"/}}
{{__ text="some text for translate"/}}
</code>



===== Атрибуты =====

 * **text** обязательный атрибут, содержащий текст для перевода.
