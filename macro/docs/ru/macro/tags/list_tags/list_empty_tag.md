# Тег {{list:empty}}
## Описание
Отображает контент в случае, если список пуст. Используется внутри тега **{{list}}**

## Синтаксис

    {{list:empty}}
    ...some content
    {{/list:empty}}

Также есть алиас **{{list:default}}**.

## Область применения
Внутри [тега {{list}}](./list_tag.md).

## Атрибуты
Нет.

## Содержимое
Любое.

## Пример использования
см. пример для [тега {{list}}](./list_tag.md).
