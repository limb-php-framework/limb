====== Тег {{list:empty}} ======

===== Описание =====

Отображает контент в случае, если список пуст. Используется внутри тега **%%{{list}}%%**


===== Синтаксис =====

<code xml>
{{list:empty}}
...some content
{{/list:empty}}
</code>

Также есть алиас **%%{{list:default}}%%**.

===== Область применения =====

Внутри [[list_tag|тега {{list}}]].

===== Атрибуты =====

Нет.

===== Содержимое =====

Любое.

===== Пример использования =====

см. пример для [[list_tag|тега {{list}}]].
