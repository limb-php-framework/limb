====== Тег {{tree:nextlevel]] ======

===== Описание =====

Тэг **%%{{tree:nextlevel}}%%** определяет место, в которое будет помещен текст, сгенерированный для дочерних узлов текущего узла.

===== Область применения =====

Внутри [[tree_node_tag|тега {{tree:node}}]].

===== Синтаксис =====

<code xml>
  {{tree:nextlevel /}}
</code>

===== Пример использования =====

см. описание [[tree_tag|тега {{tree}}]].
