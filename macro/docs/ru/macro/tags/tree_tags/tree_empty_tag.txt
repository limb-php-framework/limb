====== Тег {{tree:empty}} ======

===== Описание =====

Отображает контент в случае, если в дереве нет узлов.


===== Синтаксис =====

<code xml>
{{tree:empty}}
...some content
{{/tree:empty}}
</code>

===== Область применения =====

Внутри тега [[tree_tag| {{tree}}]].

===== Пример использования =====

см. описание [[tree_tag|тега {{tree}}]].
