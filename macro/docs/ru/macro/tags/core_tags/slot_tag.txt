====== Тег {{slot}} ======
===== Описание =====

Представляет из себя место, куда может быть вставлен контент (например другой шаблон) во время компиляции. см. [[wrap_tag|тег {{wrap}}]].

===== Синтаксис =====

<code xml>
 {{slot id='slot_name'/}}
</code>

===== Область применения =====

В любом месте MACRO шаблона.

===== Атрибуты =====

  * **id** – идентификатор тега.

===== Содержимое =====

Нет.
===== Пример использования =====

см. пример использования для тега [[wrap_tag|тег {{wrap}}]].
