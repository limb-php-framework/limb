====== Класс lmbProxy ======

**lmbProxy** - класс для реализации proxy-объектов.

lmbProxy перехватывает все обращения к аттрибутам и методам проксируемого объекта. Он проверяет, был ли оригинальный объект "отрезолвлен", то есть получена реальная копия объекта, для которого создан lmbProxy. Резолвинг оригинального объекта производится в закрытом методе 
**lmbProxy :: _createOriginalObject()**.

Для принудительного резолвинга оригинального объекта используется метод lmbProxy :: **resolve()**.

[[handles|lmbHandle]] - это пример использования lmbProxy для реализации отложенной инициализации объектов.
